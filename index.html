Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • Meta is information that isn't visible to the user but details infomation about the site.
  • The body element represents the visible content shown to the user.
  • Alt is used to add details about a image or element to help with search engines.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An external CSS style sheet is most common to seperate the code.
  • Assigning a class like "card" lets you make changes in CSS and affect all elements linked to that card.

Git

  • Git is a tool to push and pull coding work on a project
  • The project files also called a repo and are accessed on github.com
  • From the repo we find the main branch of the project
  • Developers copy the code and create secondary branches to work on indepently from the main code
  • Those branches get approved and merged with the main branch
  • git status: checks what branch we're working on
  • git checkout -b "branch name": creates a new branch to work on
